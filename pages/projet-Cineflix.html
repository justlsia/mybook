À propos du projet Cineflix

CineFlix est une application web qui permet de consulter et sauvegarder localement des fiches détaillées de films en s'appuyant sur l'API TMDB pour l'exploration de contenus cinématographiques.

Date de réalisation : Novembre 2024

Technologies : PHP, MariaDb

Environnement : Symfony

Fonctionnalités principales :

  • Connexion à l’API TMDB pour récupérer des informations sur les films
  • Visualisation des films par : genre, meilleures notes, recherche par nom de films
  • Affichage de la fiche détaillée d’un film (titre, résumé, note, image, etc..)
  • Sauvegarde locale des fiches de films dans une base de données pour consultation hors ligne

L’application permet de découvrir des films par genre, note ou recherche, et d’accéder à des détails comme les synopsis, affiches et acteurs principaux. Une fonctionnalité de sauvegarde locale permet d’enregistrer les fiches de films dans une base de données, offrant ainsi un accès même hors ligne. CineFlix utilise la flexibilité d’une API dynamique avec un stockage local.
